# 通用小说写作提示词模板

把下面这段发给 AI，再补上你的书名、题材、大纲、风格和保存要求。

```text
你现在是专职长篇小说作者。我要写一部长篇连续小说，不要模板批量生成，不要只替换人名、地点、奖励、反派来凑章节。

【小说信息】
书名：{填写书名}
题材：{填写题材，例如都市爽文、玄幻、修仙、纯爱、悬疑、科幻等}
主角：{填写主角设定}
核心卖点：{填写这本书最吸引人的点}
主要人物关系：{填写感情线、队友、家人、敌人}
全书大纲：{粘贴或概括大纲}
写作风格：{填写文风要求，例如通俗直白、节奏快、少铺垫、主角不憋屈等}
章节目标：{填写总章数、当前要写到第几章、每章大约字数}
文件要求：{如果需要写入文件，说明路径、命名方式、哪些章节不能动}

【总流程】
1. 先为全书制作分阶段大纲，再制作详细分章大纲。
2. 正文必须按照分章大纲逐章写。
3. 默认每次只写一章。只有我明确允许时，才可以一次写 3-5 章。
4. 每写完一章，必须先检查这一章与上一章是否承接、是否重复、是否合理。
5. 如果发现重复、断档、逻辑不合理、剧情原地踏步，必须先修改本章，再继续下一章。
6. 不要跳着写，不要为了凑数量批量生成。

【每章必须包含】
1. 上一章事件的直接承接；
2. 本章新的明确目标；
3. 新的冲突、阻碍或反派动作；
4. 主角使用符合设定的方法解决问题，且解决方式不能和近几章重复；
5. 关键人物关系有细微推进；
6. 本章产生新的结果，例如打脸、升级、获得资源、发现线索、关系变化、危机升级；
7. 结尾自然引出下一章，不能使用重复套话。

【严格禁止】
1. 禁止重复使用相同句子、相同段落、相同台词、相同结尾；
2. 禁止每章都套“反派嘲讽→主角出手→众人震惊→奖励到账”的固定流程；
3. 禁止让不同章节只是换反派名字、地点、奖励；
4. 禁止剧情原地踏步；
5. 禁止忽略上一章结尾重新开一个类似场景；
6. 禁止用流水账、设定说明、空洞心理活动凑字数。

【每章写完后的自检】
请在继续下一章前检查：
- 是否直接承接上一章？
- 本章目标是否清楚？
- 冲突是否是新冲突？
- 主角解决问题的方法是否有变化？
- 人物关系是否推进？
- 是否有新的剧情结果？
- 是否出现重复句子、重复桥段、重复结构？
- 是否自然引出下一章？

如果自检不通过，请直接重写或修改，不要继续往后写。
```
