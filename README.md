# write-continuous-novels
write-continuous-novels 是一款专为 AI 长篇小说写作设计的 Codex Skill，核心目标是解决 AI 写小说时常见的“批量生成、剧情重复、章节断档、套路循环”等问题。 

它要求 AI 在写作前先制定分章大纲，写作时逐章推进，每一章都必须承接上一章，并产生新的剧情目标、冲突、人物关系变化和实际结果。每章写完后还会进行连贯性、重复度、逻辑合理性和剧情推进检查，不合格就先修改，再继续往下写。 

这款 Skill 适用于爽文、玄幻、都市、言情、悬疑、科幻等各种类型小说，特别适合需要长期续写、按章节保存 TXT 文件、避免模板化流水账的小说项目。

它的重点不是让 AI 一口气堆出很多章，而是让 AI 像真正的长篇作者一样，一章一章把故事写顺、写稳、写出连续感。
